This backend powers the **TEOS Forge GitHub App**.

It performs **observation, validation, and signaling**
while preserving human authority and governance control.

---

## Responsibilities

- Receive GitHub webhooks
- Validate contribution and governance events
- Trigger audit signals
- Issue non-monetary badges
- Log governance-relevant activity

---

## Explicit Non-Responsibilities

This backend **does NOT**:
- Merge code
- Change roles
- Execute treasury actions
- Trigger on-chain execution

All authority remains human and governed.
